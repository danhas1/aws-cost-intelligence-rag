# Supported Questions

The AI Cloud Cost & Infrastructure Intelligence Assistant is designed to answer questions related to AWS infrastructure, cloud costs, storage usage, incidents, deployments, and operational efficiency.

## Cost Analysis

- Which AWS services generate the highest costs?
- Which team generates the highest AWS expenses?
- Why did AWS costs increase this month?
- Which deployment caused a cost increase?

## Storage Analysis

- Which S3 buckets consume the most storage?
- Which storage resources have not been accessed recently?
- Which team stores the most data?
- Which storage resources appear unnecessary?

## Infrastructure Analysis

- Which resources are underutilized?
- Which resources appear abandoned?
- Which services consume the most infrastructure resources?
- Which infrastructure can potentially be removed?

## Incident Investigation

- Are there incidents related to recent cost increases?
- Which resources were involved in previous outages?
- What was the root cause of a specific incident?

## Governance & Policy Validation

- Which resources violate company policies?
- Which storage resources exceed retention limits?
- Which resources require review according to governance rules?

## Deployment Analysis

- Which deployment introduced infrastructure changes?
- Which deployment increased storage usage?
- Which deployment is related to a specific incident?
